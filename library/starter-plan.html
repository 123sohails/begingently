New Muslim Starter Plan (30 Days)

A gentle month-long roadmap to build your Islamic foundation without pressure.

Focus on consistency, not perfection. If you miss a day, restart the next day. A small step done daily is better than a big plan you cannot sustain.

How To Use This Plan

  • Keep your daily learning to 20-40 minutes.
  • Use phone alarms for prayer and review time.
  • Pair each new habit with an existing one.
  • Review every Sunday and adjust gently.

Week 1: Build One Anchor

Main goal: Pray at least one prayer on time daily (Dhuhr is easiest for most people).

Daily minimum: 1 prayer + 5 minutes Qur'an translation + 1 short dua.

Checklist: Learn wudu, learn basic prayer movements, set one fixed prayer alarm.

Week 2: Add Structure

Main goal: Pray two prayers daily (Dhuhr and Asr).

Daily minimum: 2 prayers + 7-10 minutes Qur'an + basic dhikr after salah.

Checklist: Memorize Al-Fatiha meaning, set prayer corner, track prayer consistency.

Week 3: Deepen Understanding

Main goal: Reach three daily prayers and improve focus.

Daily minimum: 3 prayers + one hadith reflection + 10 minutes learning.

Checklist: Learn common phrases, begin reading short surahs with translation, ask one question to a trusted source.

Week 4: Complete The Foundation

Main goal: Build toward all five prayers and stable daily worship rhythm.

Daily minimum: 4-5 prayers + 10-15 minutes Qur'an/hadith + nightly dua.

Checklist: Plan around Fajr and Isha, keep Friday review routine, identify one supportive community contact.

30-Day Completion Check

  • I can perform wudu confidently.
  • I have a daily salah routine.
  • I read some Qur'an every day or most days.
  • I make dua in my own words regularly.
  • I know where to continue learning next.

Continue Your Journey

When this 30-day plan ends, continue with Salah, Qur'an, and daily practice pages in the library.
